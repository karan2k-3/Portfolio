Campus Marketplace (UniSwap)

A student-focused marketplace app that lets university students buy and sell items on campus using a simple, modern mobile interface.

Screenshots

About the app

The app is built with Flutter and Firebase and is designed specifically for university students. Users can sign up, create listings for items they want to sell, browse available items, and contact other students to arrange purchases on campus.

Key features

  • Email/password sign up and login with Firebase Authentication.
  • Create, edit and delete listings with images, description and price.
  • Real-time listing updates stored in Cloud Firestore.
  • Simple search and browse experience for campus items.
  • Clean red and black UI designed for Android phones.

Firebase features

  • Firebase Authentication for secure student sign-in.
  • Cloud Firestore for real-time listing and user data.
  • Firebase Storage for uploading and serving item images.
  • Security rules restricting writes to authenticated users.

Tech stack

  • Flutter and Dart for the mobile app.
  • Firebase Authentication for user accounts.
  • Cloud Firestore for storing listings and user data.
  • Firebase Storage for item images.
